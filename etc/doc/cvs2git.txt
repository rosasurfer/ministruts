How to convert and upload a CVS repository to GitHub
---------------------------------------------------- 

 • collect authors:  cvs -d $CVSROOT log | egrep -o "author[^;]+;" | sort -u 

 • copy /usr/share/doc/cvs2svn-2.3.0/cvs2git-example.options  cvs2git.options 

 • create the cvs2git working directory specified in:
   ctx.tmpdir = r'cvs2git-tmp'

 • change:
   all CVSTextDecoder()             to       utf8, latin1
   run_options.set_project          to       r'/var/cvs/{repo_directory}/{repo_name}'
   DefaultEOLStyleSetter(none)      to       DefaultEOLStyleSetter('LF')
   author_transforms = {...}

 • launch conversion:
   cvs2git --options=cvs2git.options > cvs2git.log 

 • create git repository:
   mkdir {project_name} && cd {project_name} && git init .

 • import files created by cvs2git:
   cat ../cvs2git-tmp/git-{blob,dump}.dat | git fast-import

 • cleanup and repack repo:
   git gc --aggressive --prune=now
   git repack -a -d -f
   
 • add remote:
   git remote add origin git@github.com:{user_name}/{repo_name}.git

 • mirror to github:
   git push --mirror [-n]

 • checkout working copy:
   git reset --hard

 • set upstream branch
   git branch --set-upstream-to=origin/master master

 • migrate .cvsignore to .gitignore files
